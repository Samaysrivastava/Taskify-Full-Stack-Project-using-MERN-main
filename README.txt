# Taskify 🚀  
### Smart Team Collaboration & Task Management Platform

A modern full-stack team collaboration and task management platform built with the MERN Stack. Taskify helps teams organize workflows, assign tasks, track progress, and collaborate efficiently in real time.

---

# 🌐 Overview

Taskify is a cloud-based project and task management application designed for modern teams and organizations. It provides a centralized workspace where admins and team members can manage tasks, communicate, monitor progress, and improve productivity.

The platform is built using scalable and modern technologies including MongoDB, Express.js, React, Node.js, Redux Toolkit, Tailwind CSS, and Headless UI.

Whether you're managing a startup team, college project, or remote workforce, Taskify simplifies task tracking and collaboration.

---

# ✨ Features

## 👨‍💼 Admin Features

### 🔹 User Management
- Create and manage admin accounts
- Add new team members
- Activate or disable user accounts
- Delete users permanently

### 🔹 Task Management
- Create tasks and assign them to one or multiple users
- Update task details and deadlines
- Manage task lifecycle

### 🔹 Task Status & Priority
- Mark tasks as:
  - Todo
  - In Progress
  - Completed

- Set priority levels:
  - High
  - Medium
  - Normal
  - Low

### 🔹 Subtasks Support
- Break large tasks into smaller manageable subtasks

### 🔹 Asset Upload
- Upload images and other task-related files

### 🔹 Task Control
- Trash or permanently delete tasks

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/Samaysrivastava/Taskify-Full-Stack-Project-using-MERN-main.git

cd taskify
```

---

## 2️⃣ Setup Backend

```bash
cd server
npm install
```

### Create `.env` file

```env
PORT=5000
MONGODB_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```

### Run Backend

```bash
npm run dev
```

---

## 3️⃣ Setup Frontend

```bash
cd client
npm install
npm run dev
```
